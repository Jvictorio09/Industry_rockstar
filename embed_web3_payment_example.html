Web3 Payment Widget - Embed Example

This page demonstrates how to embed the Web3 payment widget in an iframe.

Note: Replace YOUR_DOMAIN with your actual domain where the payment widget is hosted.

Live Preview

Embed Code

<iframe id="web3-payment-iframe" src="https://YOUR_DOMAIN/payment/" title="Web3 Payment Widget" scrolling="no" style="min-height: 600px; width: 100%; border: none;"> </iframe> <script> // Auto-resize iframe based on content window.addEventListener('message', function(event) { if (event.data.type === 'web3-payment-height') { const iframe = document.getElementById('web3-payment-iframe'); if (iframe) { iframe.style.height = event.data.height + 'px'; } } }); </script>

Features

  • ✅ Auto-resizing iframe (adjusts to content height)
  • ✅ Secure payment processing via MetaMask
  • ✅ Supports Course Payment ($4,995), Supplier Listing (100 USDC), and Custom Amounts
  • ✅ Real-time transaction status updates
  • ✅ Mobile responsive

Customization

You can customize the iframe styling:

<style> #web3-payment-iframe { width: 100%; max-width: 800px; border: 2px solid #e0e0e0; border-radius: 12px; box-shadow: 0 4px 6px rgba(0,0,0,0.1); } </style>
